NUMBER GAME

🎯 Jogo de Adivinhação de Números
Um projeto desenvolvido com HTML, CSS e JavaScript que desafia o jogador a descobrir um número secreto gerado aleatoriamente pelo sistema.

 
 Sobre o projeto

Este jogo foi criado para praticar conceitos fundamentais de desenvolvimento web, incluindo manipulação do DOM, lógica de programação, interação com o usuário e estilização de interfaces.

O objetivo é simples: o jogador deve tentar adivinhar o número secreto e receberá dicas para encontrar a resposta correta.

🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

🎮 Funcionalidades

* Geração aleatória de número secreto
* Validação das tentativas do usuário
* Dicas indicando se o número é maior ou menor
* Mensagens de acerto e erro
* Interface simples e intuitiva

📚 Aprendizados

Durante o desenvolvimento deste projeto, pratiquei:

* Estruturas condicionais
* Funções em JavaScript
* Manipulação de elementos HTML
* Eventos de interação do usuário
* Organização de código e boas práticas

🔗 Objetivo

Este projeto faz parte da minha jornada de aprendizado em programação e desenvolvimento web, com foco em fortalecer a lógica de programação e criar aplicações interativas.
